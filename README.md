# react-native-super-awesome

React native module for SuperAwesome

## Installation

```sh
npm install react-native-super-awesome
```

## Usage

```js
import { SuperAwesomeView } from "react-native-super-awesome";

// ...

<SuperAwesomeView color="tomato" />
```
